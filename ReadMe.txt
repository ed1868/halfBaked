Step 1. 
Ensure you have React native setup completed on your machine before running below commands, please follow this link for further details :
https://facebook.github.io/react-native/docs/getting-started

HalfBaked, is built using react native CLI and does not use expo.

To run IOS, you will require the project to run on Mac only as IOS apps can only be developed on a Macbook.

Step 2:
Once you have a working dev environment, navigate inside HalfBaked folder using your terminal or command prompt and run below command : "yarn install"
This will install all the required node dependencies in your project.

Step 3:
Once you have installed all dependencies run below command to link icons :
"react-native link react-native-vector-icons"


Step 4 :
To run application run command : "react-native run-android"

This shall run app in the simulator.
If you wish to run on your own device, please refer below link :
https://facebook.github.io/react-native/docs/running-on-device
